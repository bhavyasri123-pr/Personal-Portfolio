# Personal Portfolio Website

A full-stack personal portfolio website built using React.js, Node.js, Express.js, and MySQL.

## Features

- Responsive portfolio website
- Projects section (data from MySQL)
- Certificates section (data from MySQL)
- Contact form (messages stored in MySQL)
- Admin Login
- Admin Dashboard
- Manage Projects (Add, Edit, Delete)
- Manage Certificates (Add, Edit, Delete)
- Manage Contact Messages

## Tech Stack

### Frontend
- React.js
- Bootstrap
- Axios

### Backend
- Node.js
- Express.js

### Database
- MySQL
